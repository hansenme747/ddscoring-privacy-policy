DDScoring Privacy Policy

Last updated: March 17, 2026

This Privacy Policy explains how DDScoring collects, uses, stores, and shares data when you use the app.

1. Data We Collect

DDScoring may store and process the following data:

  • Profile data: display name, player record identifier, optional avatar selection
  • Gameplay data: match records, per-game scores, sink stats, self-sinks, death beers, match outcomes, and rating data (for example, DDPR and match counts)
  • Social data: friend requests, accepted friend links, and public friend code/profile discovery data
  • Technical data: basic CloudKit record metadata required for sync

2. Where Data Is Stored

DDScoring uses Apple iCloud / CloudKit for data storage and sync.

  • Private CloudKit database: profile-specific/private data tied to your iCloud account
  • Public CloudKit database: shared/public gameplay and ranking data used for cross-user features
  • Local on-device storage: local cache and offline-first sync behavior

3. How We Use Data

  • Create and maintain your profile
  • Save and sync match history across your devices
  • Enable friends/followers features
  • Compute and display rankings, leaderboards, and player statistics
  • Support offline usage and delayed sync when connectivity returns

4. Data Sharing

  • Data written to the public CloudKit database can be visible to other app users as part of shared features (for example, public rankings/leaderboards and shared match participation records)
  • Data in your private CloudKit database is not publicly visible
  • We do not sell personal data

5. Third Parties

DDScoring relies on Apple services (CloudKit/iCloud) to store and sync data. Apple may process data according to its own terms and privacy policies.

6. Data Retention

Data is retained as long as needed for app functionality unless removed by the developer or user through app/account data changes.

7. Your Choices

  • Stop using the app at any time
  • Control iCloud usage via your Apple ID/iCloud settings
  • Request deletion/correction of app-managed profile data by contacting us

8. Children

DDScoring is not directed to children under 13.

9. Changes to This Policy

We may update this policy. Updates will be posted at this URL with a revised "Last updated" date.

10. Contact
